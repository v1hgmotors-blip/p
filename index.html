📋 설치 방법 (딱 2단계!)
✅ STEP 1 — 스크립트 다운로드
위 ⬇️ PowerShell 스크립트 다운로드 버튼 클릭
→ 키불출_PC알림.ps1 파일을 바탕화면에 저장
▶️ STEP 2 — 파일 실행 (끝!)
바탕화면의 키불출_PC알림.ps1 파일
→ 우클릭 → "PowerShell로 실행" 클릭
→ "이 앱이 변경하도록 허용하시겠습니까?" → 예 클릭
→ 화면 오른쪽 아래에 ✅ 서버 시작! 알림 확인
💡 관리자 권한·방화벽 설정이 자동으로 처리됩니다.
별도로 명령어를 입력할 필요가 없습니다!
📱 STEP 3 — 모바일에서 PC IP 입력
PC에서 Windows키 + R → cmd → 확인
검은 창에 아래 입력 후 Enter:
ipconfig | findstr IPv4 📋 탭하면 복사
→ 192.168.x.xxx 형태의 숫자가 PC IP 주소
→ 위 PC 목록에 IP + 포트(9999) 추가 → 💾 저장
→ 📤 전체 테스트로 알림 확인!
⚠️ "실행 오류" 뜰 때 (최초 1회만)
PowerShell 실행 정책 오류가 나오면:
Windows 검색 → PowerShell → 관리자 권한으로 실행
아래 명령어 입력 후 Enter → Y 입력 → Enter
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser 📋 탭하면 복사
→ 이후 다시 파일 우클릭 → PowerShell로 실행
🚀 PC 켤 때 자동 시작 설정 (선택)
Windows키 + R → shell:startup → 확인
→ 열린 폴더에 키불출_PC알림.ps1 복사·붙여넣기
→ 이후 PC 켤 때마다 자동 실행 🎉
